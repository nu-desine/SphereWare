# SphereWare #

SphereWare is the AlphaSphere and AlphaSphere Elite Firmware. 

## Hardware ##

AlphaSphere uses an Atmel Atmega32u4 MCU clocked at 16Mhz.

## Requirements ##
To build this software you will need the avr-gcc toolchain.
